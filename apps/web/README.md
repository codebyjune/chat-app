# Chat App

Vue 3 聊天前端，负责登录注册、好友搜索、好友管理、会话展示与实时消息交互。

## 环境变量

复制 `.env.example` 为 `.env`：

```env
VITE_API_BASE_URL=http://localhost:3000
```

## 脚本

- `pnpm dev`
- `pnpm build`
- `pnpm preview`

## 当前功能

- 登录与注册
- 用户资料拉取与鉴权状态恢复
- 搜索用户并发送好友申请
- 查看待处理申请并接受或拒绝
- 查看好友列表与删除好友
- 会话按最近消息排序并展示未读消息数
- 按好友名或最近消息内容搜索会话
- 支持分页加载更早的历史消息
- 支持在线状态展示与消息已读标识
- 拉取历史会话并通过 Socket.IO 实时收发消息
